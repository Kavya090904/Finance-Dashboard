# FinTrack - Smart Personal Finance Assistant (Zorvyn)

[![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-B73BFE?style=for-the-badge&logo=vite&logoColor=FFD62E)](https://vitejs.dev/)
[![Framer Motion](https://img.shields.io/badge/Framer_Motion-0055FF?style=for-the-badge&logo=framer&logoColor=white)](https://www.framer.com/motion/)

A premium, AI-powered personal finance management ecosystem built with **React**, **Recharts**, and **Framer Motion**. This project elevates the standard finance dashboard into a gamified, insightful, and highly interactive experience for modern users.

---

## ✨ Premium & Unique Features

### 🏦 Advanced Financial Hub
- **Bento Grid Architecture**: A modular, data-dense layout that adapts seamlessly to any screen size.
- **AI Executive Summary**: A top-level "Human-Readable" status report that translates numbers into actionable financial advice.
- **Financial Persona (Gamified)**: Analyzes spending habits to assign archetypes like "Strategic Saver" and ranks your performance in the **85th Percentile** of savers.

### 📊 Deep-Dive Analytics
- **Multi-Line Historical Tracking**: Area charts comparing **Income vs. Expenses** over 6 months to visualize net worth growth.
- **50/30/20 Budget Ruler**: Real-time tracking of Needs, Wants, and Savings against professional financial standards.
- **Merchant Leaderboard**: Automatic identification of top spending brands (Amazon, Zomato, etc.) with color-coded brand progress bars.
- **Category Radar Profile**: A 360-degree spider-chart of your monthly spending behavior.

### 🛠️ High-Performance Transactions
- **Digital Receipt Reconciliation**: Visual "PDF/Image" indicators next to expenses to simulate a verified audit trail.
- **Real-Time Data Engine**: Instant search, multi-column sorting, and category filtering across hundreds of entries.
- **Smart Data Export**: Professional-grade "Download Report" utility for financial reconciliation.

### 🎮 Gamification & Engagement
- **12-Day Tracking Streak**: A visual incentive badge for consistent daily monitoring.
- **Savings Trophies**: Milestone achievemets like "Budget King" and "Daily Pulse" with animated unlock states.
- **Investment Goal Planner**: A predictive tool where users enter a goal (e.g., ₹5L) and the system calculates the exact **Target Completion Date**.

### 🎨 High-End Aesthetics
- **3D Interactive Virtual Card**: A glassmorphism credit card with real-time balance updates and a 3D-tilt hover effect.
- **Live Market Ticker**: A persistent scrolling footer showing real-time market sentiment and financial tips.
- **Breathing AI Assistant**: An animated floating chatbot avatar that provides proactive insights and a reactive typing UI.
- **Dark/Light Mode Persistence**: Seamless theme switching with absolute state memory.

---

## 🛠️ Tech Stack

- **Framework**: React 18 (Vite)
- **Styling**: Vanilla CSS (Custom tokens, Flex/Grid)
- **Visuals**: Recharts (High-performance SVG charts)
- **Motion**: Framer Motion (Orchestrated stagger animations)
- **Icons**: Lucide React (Premium iconography)
- **State**: React Context API & `localStorage`

## 📦 Getting Started

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Kavya090904/Finance-Dashboard.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
